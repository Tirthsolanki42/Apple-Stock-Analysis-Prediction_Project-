# 📈 Stock Price Analysis & Prediction Dashboard

An end-to-end data science project that analyses stock market data 
and predicts next-day price movement using Machine Learning.

## 🔧 Tech Stack
- Python, Pandas, Scikit-learn
- Streamlit (interactive dashboard)
- yFinance (real stock data)
- Matplotlib (visualisation)

## 📊 Features
- Live stock data for any ticker (AAPL, TSLA, GOOGL etc.)
- Moving average analysis (50-day & 200-day)
- Random Forest ML model to predict next-day price direction
- Model accuracy score & feature importance chart
- Interactive sidebar to change stock and date range

## 🚀 How to Run
1. Clone this repo
2. Install dependencies: pip install -r requirements.txt
3. Run: streamlit run stock_dashboard.py

## 📚 Background
Built as a portfolio project following MSc Business Analytics 
(University of Greenwich), demonstrating end-to-end data science 
skills including data collection, feature engineering, ML modelling 
and dashboard deployment.
